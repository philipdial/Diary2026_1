This diary uses Firebase (free) for live sync across all phones and computers. Do this once, then share the link with everyone.
1Go to console.firebase.google.com → Add project → name it (e.g. "diary2026") → disable Analytics → Create.
2In left menu: Build → Realtime Database → Create database → choose region → Start in test mode → Enable.
3Project Settings (gear icon) → scroll to "Your apps" → click </> Web → register app → copy the firebaseConfig object → paste below.
